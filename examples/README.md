# Examples

This directory contains example components generated by the CLI tool.

## TestButton (JSX)
A simple JSX component with CSS styles.

```bash
gen-component create TestButton -t jsx -s -p ./examples
```

## Card (TSX)
A TypeScript React component with CSS modules.

```bash
gen-component create Card -t tsx -s -p ./examples
```

## Testing the CLI

You can test the CLI with various options:

```bash
# Basic JSX component
gen-component create MyComponent

# TypeScript component
gen-component create MyComponent -t tsx

# Component with styles
gen-component create MyComponent -s

# Custom path
gen-component create MyComponent -p ./custom/path

# All options combined
gen-component create MyComponent -t tsx -s -p ./src/components
```

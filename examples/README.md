# Examples

This directory contains example components generated by the CLI tool, showcasing all available templates.

## ðŸ“¦ Available Examples

### TestButton (JSX - Functional Component)
A simple JSX functional component with CSS styles.

```bash
gen-component create TestButton -t jsx -s -p ./examples
```

### Counter (TSX - Class Component)
A TypeScript class component with lifecycle methods and CSS modules.

```bash
gen-component create Counter -c -t tsx -s -p ./examples
```

### useFetch (TSX - Custom Hook)
A custom React hook following TypeScript best practices.

```bash
gen-component create useFetch -k -t tsx -p ./examples
```

### Auth (TSX - Context Provider)
A Context API provider with a custom hook for state management.

```bash
gen-component create Auth -x -t tsx -p ./examples
```

### OptimizedList (TSX - Memoized Component)
A React.memo optimized component for performance.

```bash
gen-component create OptimizedList -m -t tsx -s -p ./examples
```

### withAuth (TSX - Higher Order Component)
A HOC pattern for adding authentication logic.

```bash
gen-component create withAuth -H -t tsx -p ./examples
```

### LoginForm (TSX - Form Component)
A form component with validation logic.

```bash
gen-component create LoginForm -f -t tsx -s -p ./examples
```

### UserService (TSX - API Service)
An API service with CRUD operations.

```bash
gen-component create UserService -a -t tsx -p ./examples
```

### StyledButton (TSX - Styled Component)
A component using styled-components.

```bash
gen-component create StyledButton --styled -t tsx -p ./examples
```

## ðŸ§ª Testing All Template Types

You can test the CLI with various options:

```bash
# Functional components
gen-component create MyComponent              # JSX functional
gen-component create MyComponent -t tsx       # TSX functional

# Class components
gen-component create MyComponent -c           # JSX class
gen-component create MyComponent -c -t tsx    # TSX class

# Custom hooks
gen-component create useMyHook -k             # JSX hook
gen-component create useMyHook -k -t tsx      # TSX hook

# Context providers
gen-component create MyContext -x             # JSX context
gen-component create MyContext -x -t tsx      # TSX context

# Memoized components
gen-component create MyList -m -t tsx -s      # Memoized component

# Higher Order Components
gen-component create withLogger -H -t tsx     # HOC

# Form components
gen-component create ContactForm -f -t tsx -s # Form with validation

# API services
gen-component create ProductService -a -t tsx # API service

# Styled components
gen-component create MyButton --styled -t tsx # Styled component

# With styles
gen-component create MyComponent -s           # With CSS
gen-component create MyComponent -t tsx -s    # With CSS Module

# Custom path
gen-component create MyComponent -p ./custom/path

# All options combined
gen-component create MyComponent -c -t tsx -s -p ./src/components
```
